Multimodal UAV Simulation Benchmark

Designed and implemented the complete multimodal simulation and data-generation pipeline for a 3D UAV perception benchmark built on the UE4-based CARLA engine. This pipeline integrates realistic UAV flight dynamics, synchronized multi-sensor acquisition, cross-modal timestamp alignment, and unified coordinate transformation, enabling large-scale generation of geometrically consistent RGB, LiDAR, radar, and DVS data across diverse scenes and weather conditions. Depression Severity Estimation System

Built a deep learning system that estimates depression severity from facial video using OpenFace-based landmark tracking. From HDR (Histogram of Displacement Range) and FAU (Facial Action Unit) features, I trained a DNN to predict severity levels. I also developed a cross-platform web system that records user videos and performs server-side analysis for longitudinal mental health monitoring.
